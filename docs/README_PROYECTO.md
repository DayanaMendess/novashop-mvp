# NovaShop MVP - Descripción del Proyecto

## 1. Descripción general

NovaShop MVP es una solución web básica para simular el flujo de compra de una tienda virtual.  
Su propósito es demostrar una arquitectura organizada con frontend, backend, base de datos y contenedores Docker.

## 2. Objetivo general

Desarrollar un sistema funcional de carrito de compras que permita administrar clientes, productos, carrito y pedidos, aplicando buenas prácticas de organización del proyecto.

## 3. Objetivos específicos

- Construir una API REST con Spring Boot.
- Diseñar una interfaz web moderna y distinta al ejemplo de referencia.
- Modelar la base de datos en PostgreSQL.
- Contenerizar la solución con Docker.
- Documentar la arquitectura, los ambientes y la ejecución.

## 4. Alcance funcional

El sistema cubre:

- registro de clientes
- consulta de productos
- creación de carrito
- agregado de productos al carrito
- confirmación de compra
- consulta de pedidos

## 5. Valor académico

Este proyecto sirve como evidencia de:

- organización de un repositorio profesional
- trabajo por capas
- separación entre frontend y backend
- uso de documentación técnica
- manejo de ambientes y despliegue local con Docker
